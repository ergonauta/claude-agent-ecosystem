# Claude Agent Ecosystem

A structured multi-agent workflow system for delivering software features with precision,
traceability, and human oversight at every critical decision point.

## Philosophy

> "The plan is the product. The code is the proof."

This ecosystem exists to eliminate the most common failure mode of AI-assisted development:
Claude implementing something plausible but misaligned, then building tests that validate
the wrong behaviour. Every workflow here forces alignment *before* implementation begins.

The human owns the intent. Claude owns the execution. The plan file owns the state.

---

## Core Principles

1. **Specification before implementation** — Ambiguity is resolved in the plan, not in code
2. **TDD as alignment tool** — RED tests are a contract, not a formality
3. **Small atomic blocks** — Each phase is small enough to commit, review, and verify independently
4. **Human in the loop** — Explicit approval gates at every critical transition
5. **State survives context** — The plan file is the source of truth across all context clears
6. **ADR-driven decisions** — Every non-obvious choice is recorded with its rationale

## Quick Start

### First time in a new project

```bash
# 1. Run the cartographer to map the codebase
/repo-cartographer

# 2. Start a new feature
/implementation-plan
```

### Starting a feature

1. Run `/implementation-plan` — Claude enters planning mode and challenges your requirements
2. Review and approve `implementation-plan.md` — this is your contract
3. Claude creates `implementation-status.md` — this tracks live state
4. Work through phases with approval gates (see `docs/workflow-detailed.md`)

---

## The Two Files

| File | Purpose | Who writes | Who mutates |
|---|---|---|---|
| `implementation-plan.md` | Intent, spec, acceptance criteria, phase definitions | Human (with Claude challenging) | Frozen after approval |
| `implementation-status.md` | Current phase, completed work, decisions made, ADR notes | Claude (after each phase) | Updated after each human approval |
